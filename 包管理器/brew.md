# HomeBrew

macOS 缺失的软件包管理器

[HomeBrew](https://brew.sh/index_zh-cn.html)

# 查看版本

brew --version

#

brew --help

# 升级 HomeBrew

brew update

# 安装包

brew install [package]

brew install [package] [package]

# 检测所有可升级的包

brew outdated

# 升级所有可升级的包

brew upgrade

brew upgrade [package]

# 清除缓存

brew cleanup
